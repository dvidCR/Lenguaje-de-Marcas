Resumenes de 1º de DAM

(Desarrollo de Aplicaciones Multiplataforma)

Base de Datos

En esta asignatura aprenderas a crear y gestionar tu propia base de datos y crear y leer diagramas. De primeras aprenderas a comprender y crear diagramas para entender como va tomar forma la base de dato, una vez que el profesor estime que saben lo suficiente pasan directamente al MySQLWorbench para crear el script de la base de datos.
Más adelante tambien te lo pueden dar al reves, basicamente tu creas el diagrama y explicas el porque a partir de un script que te haya pasado el profesor.

Lenguaje de Marcas

En esta asignatura aprenderas a que crear páginas web como esta (pero más bonita), con los lenguajes HTML, CSS y JavaScript. De primeras la asignatura tira fácil enseñandote lo más básico de HTML para daros paso a CSS, el lenguaje "complentario"(aunque podria decirse obligatorio) para HTML ya que con este podras darle estilo a tu página.
Por el camino se verá un poco de BOOTSTRAP para eseñarte distintas posibilidades ingeniosas y complicadas de estilos.
Y ya por último vereis JavaScript, con el podreis hacer toda la parte lógica de la página web.

Programación

En esta asignatura aprendereis a crear programas en Python y/o Java. La parte más resumida es que lo mejor sería que se empezara con python ya que es más sencillo y fácil de comprender, aprendereis a crear, usar y heredar clases. Una vez que más o menos hayais pasado de tirmestre y ya hayais logrado aprender muhco en python pasareis directamente a Java, el lenguaje de los objetos, ya que casi no se puede programar otra cosa.

Sistemas Informáticos

En esta asignatura aprendereis todo sobre el software de tu ordenador y un poco del hardware. Basicamente gran parte del primer trimestre os dedicareis a aprender de todo sobre el hardware de un ordenador y más o menos a montarlo.
Después de acabar esto nos dedicaremos a manejar tanto en lo windows y linux server, principalmente desde máquinas virtuales, las cuales daremos de nivel uno y nivel tres.
Otras características es que también sabremos utilizar docker.
